# 账单系统 - 本地调试展示页

Vue 3 + Vite 搭建的本地调试前端，用于调用后端 API 查看产品、订单、账单并执行创建订单、退款、还款等操作。

## 环境

- Node.js 18+
- 后端服务已启动在 **http://localhost:8080**

## 安装与运行

```bash
cd frontend
npm install
npm run dev
```

浏览器访问 **http://localhost:5173**。

## 说明

- **当前用户**：顶部输入 `userId`（如 `u1`），订单列表、账单本期/历史均按该用户查询。
- **产品**：展示产品列表，可刷新。
- **订单**：可创建订单（必填订单号、卡号、金额、userId）；列表支持按用户刷新；可对正常订单发起退款。
- **账单**：可查「本期应还」「历史账单」；对未结清/逾期账单可输入金额后点击还款。

前端通过 Vite 代理将 `/api` 转发到 `http://localhost:8080`，无需单独配置 CORS 即可本地联调。
